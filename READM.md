Curso Java Intermediário
Status: CONCLUÍDO
Repositório dedicado aos estudos, desafios e exercícios desenvolvidos durante o "Curso de Java Intermediário", ministrado por Professora Loiane Groner . 
O foco foi aprofundar os conhecimentos na linguagem Java, passando de conceitos básicos para a construção de aplicações mais robustas e orientadas a objetos.
Tecnologias e Conceitos Dominados
Durante o treinamento, explorei e consolidei os seguintes tópicos:
Java Intermediário

•	Enumeradores(Enum): Enumeradores com Classe(construtor e métodos), Métodos Value e ValueOf, Métodos abstratos e por último fiz um calculadora com o Enum.

•	Mais Elementos básicos da Linguagem Java: Wrappers, Autoboxing e Unboxing, Static Import, Escopo de Variáveis, Passagem de Parâmetros por Valor e por
Referencia, Vararg, Printf, Classes aninhadas, Annnotations, Garbage Collector.

•	Programação Multi – tarefa|Threads: Criando Threades mais métodos start, run e sleep, Interface Runnable, Métodos IsAlive e Join, Threads: definindo
prioridades, Synchronized, Notify,Wait e NotifyAl, Resume, Suspend e Stop, Deadlock e foi feito um semáforo com Threads.

•	Trabalhando com Strings: Construtores, concatenação, Comparação de String(equalsIgnorCase, regionMatches, startWht, endswht, compareTo), Extraindo
chars(charAr, getChars, getBytes, toCharArray), Modificando uma String(substring, concat, replace, trim), Modificando letras(toLowerCase, toUpperCase), Juntando
e separando String(join, split), Concatenação com StringBuffer e StringBuilder,String Tokinizer.

•	Classes Utilitárias| Datas, Locale, Math: Classe Math, Classe Calender, Classe Date, GregorianCalendar, Formatando datas com SimpleDateFormat, 
Locale(internacionalização), Formatando datas com DateFormat e Locale, Time Zone, Formatando números(NumberFormat e DecimalFormat), 
trabalhando com BigDecimal e BigInteger), Random, Datas no Java 8+.
